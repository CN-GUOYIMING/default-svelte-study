# README

## Svelte app を作成

```shell
# プロジェクトを新規作成
npx degit sveltejs/template app-name

# プロジェクトディレクトリに入る
cd app-name

# 依存をインストール
# 依存定義ファイル：app-name/package.json
npm install

# ローカルサーバーを起動
npm run dev
```

- [svelte document (CN)](https://www.sveltejs.cn/)
